DELETE ALL CONVERGED RECORDINGS IN YOUR ORG BEFORE A CUTOFF DATE

⚠ This permanently deletes every converged recording in your organization (listed via GET /v1/admin/convergedRecordings, which requires Admin orCompliance Officer role) that was created on or before the cutoff date. The script lists in 30-day windows (the admin endpoint requires the interval between from and to to be within 30 days) and deletes each item until the list is empty, then walks the window back one more month (up to months_back windows).

Bulk Delete All Converged Recordings Before Cutoff (Warning: it will permanently delete all converged recordings before the cutoff date)

The bulk button uses POST /v1/convergedRecordings/softDelete and POST /v1/convergedRecordings/purge — each call handles up to 100 recordings at once, so it is dramatically faster than the one-by-one DELETE. It first moves recordings to the recycle bin, then immediately purges them from the recycle bin.
